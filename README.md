#Project Download
```
git clone https://github.com/NoFunny/ChooseLang.git
```

#Program assembly

###With tests
```
sudo gradle build
```

###Without tests
```
sudo gradle build -x test
```
    
#Program launch
```
gradle bootRun
```
###Gradle version 4.5 and higher
